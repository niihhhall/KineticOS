## 🌟 Overview
**KineticOS** is a premium, all-in-one Notion operating system designed for freelancers, solopreneurs, and small agencies to scale their operations to $10k+ months. It replaces fragmented tools with a unified workspace built for high velocity.

### Why KineticOS?
- **Replace 8+ Subscriptions**: Consolidate CRM, Project Management, Finance, and Content Planning.
- **Save $8,000+**: Drastically reduce onboarding and tool costs.
- **One-Click Resets**: Built-in "Reset Button" for weekly productivity cycles.
- **AI-Powered**: Includes a specialized chat assistant with deep knowledge of the KineticOS ecosystem.

## 🛠️ Technology Stack
- **Frontend**: Vite + React + TypeScript + Tailwind CSS
- **Backend (Edge)**: Node.js (Vercel Functions inspired)
- **Database**: Supabase (PostgreSQL with pgvector for AI embeddings)
- **Payments**: Stripe & Razorpay Integration
- **AI/LLM**: Groq (Llama 3) & Hugging Face (Embeddings)
- **Analytics**: PostHog
- **Email**: Resend

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Niihhhall/KineticOS.git
   cd KineticOS
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   ```bash
   cp .env.example .env
   ```
   *Edit `.env` and add your API keys (Stripe, Razorpay, Supabase, etc.).*

4. **Run the Development Server:**
   ```bash
   npm run dev
   ```

5. **Start the Local API Server:**
   ```bash
   # In a separate terminal
   node local-api.js
   ```

## 🔒 Security & Privacy
We take security seriously. All API keys and sensitive credentials are managed via environment variables and are **never** committed to the repository. 

> [!WARNING]
> Never share your `.env` file or commit it to a public repository. A `.env.example` is provided for reference.
